# 📝 TaskMaster — Task Manager

A beautiful, modern, Apple-style task management web app built with **pure HTML, CSS, and JavaScript**.  
Designed for smooth UX, glassmorphism UI, and mobile-first experience.

---

## ✨ Features

- 📌 Add / Edit / Delete tasks
- ✅ Mark tasks as completed
- ⭐ Favorite important tasks
- 🔍 Real-time search
- ↕️ Sorting (Newest, Oldest, A–Z, Z–A)
- 📱 Fully responsive (mobile + desktop)
- 🪟 Modern glassmorphism UI
- 💾 LocalStorage support (data saved in browser)
- 🎯 Sidebar + Bottom navigation (iOS-style)
- 🎬 Smooth animations & transitions

### ⌨️ Keyboard Shortcuts
- `Ctrl/Cmd + K` → New task
- `Ctrl/Cmd + F` → Search tasks
- `Esc` → Close modal
- `Delete` → Remove first visible task

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Glassmorphism + Responsive Design)
- Vanilla JavaScript (No frameworks)
- LocalStorage API
